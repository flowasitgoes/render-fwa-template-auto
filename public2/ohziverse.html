OHZIVERSE

Tech Specs
Frontend
  • OffscreenCanvas API for parallel rendering.
  • Web Workers for parallel processing.
  • WebRTC and UDP protocol for multiplayer and real-time communication (using geckos.io).
  • In-house Transition System for smooth view transitions.
  • WebXR API for AR experiences.
WebGL
  • Selective Bloom as post processing.
  • Pathfinding Algorithm for terrain traversal.
  • Stencil Masks for portal effects.
  • Instancing for geometry reuse and drawcall optimizations.
  • Flowmap Techniques for water flow direction.
  • Custom Lighting Shader for environment and objects.
